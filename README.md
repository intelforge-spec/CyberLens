  # 🛡️ CyberLens

**CyberLens** is an AI-powered Security Operations Center (SOC) and Threat Intelligence platform designed to help security teams detect, investigate, and respond to cyber threats from a single unified dashboard.

The platform integrates threat intelligence, IOC analysis, email analysis, AI-assisted investigation, and security news aggregation to improve security operations and accelerate incident response.

---

## ✨ Key Features

### 🧠 AI Security Assistant
- AI-powered threat explanation
- IOC interpretation
- Incident summary generation
- Security recommendations
- Threat investigation assistance

### 🔍 IOC Analyzer
- IP Address Reputation
- Domain Reputation
- URL Analysis
- File Hash Lookup (MD5, SHA1, SHA256)
- IOC Enrichment

### 📧 Email Analyzer
- Email Header Analysis
- SPF Validation
- DKIM Validation
- DMARC Validation
- Phishing Detection
- Sender Reputation

### 🌐 URL Analyzer
- Website Reputation
- SSL Certificate Information
- Domain WHOIS
- DNS Records
- Threat Intelligence Lookup

### 📰 Threat Intelligence
- Latest Cybersecurity News
- CVE Database
- Threat Actor Profiles
- Malware Intelligence
- Attack Campaign Tracking
- Threat Trends

### 🎯 Threat Hunting
- IOC Search
- Threat Correlation
- MITRE ATT&CK Mapping
- Timeline Analysis
- Investigation Workspace

### 📊 Dashboard
- Security Overview
- Threat Statistics
- Severity Distribution
- Detection Trends
- Incident Summary
- Audit Logs

### ⚙️ Administration
- User Management
- Role-Based Access Control (RBAC)
- API Integration
- System Configuration
- Audit Logging
