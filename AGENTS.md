# Translation & Terminology Rules

To maintain high-quality, professional Indonesian SOC (Security Operations Center) documentation, threat intelligence summaries, and dashboard feeds:

## Translation Guidelines
- **DO NOT** perform literal word-for-word translations from English.
- Always prefer natural, precise, and professional Indonesian cybersecurity terminology over literal transfers.

## Prohibited Terminology (Avoid)
- "eksploitasi aktif liar"
- "kampanye ancaman"
- "aktor ancaman"
- "serangan di alam liar"
- "eksploitasi di alam liar"
- "kampanye phishing"
- "musuh siber"
- "aktor jahat"

## Preferred Terminology (Use Instead)
- **eksploitasi aktif** / **sedang dieksploitasi** / **telah teridentifikasi dieksploitasi**
- **kelompok ancaman** (for threat actors / groups)
- **aktivitas ancaman** (for campaign / threat activities)
- **operasi serangan** (for campaign / attack operations)
- **pihak tidak berwenang** / **pelaku ancaman** (for bad actors / threat actors)

## Contoh Penerapan Praktis (Practical Translation Examples)
- **Salah (Avoid):** "Cisco memperingatkan eksploitasi aktif liar atas celah keamanan."
- **Benar (Good):** "Cisco memperingatkan adanya eksploitasi aktif terhadap kerentanan keamanan tersebut."
- **Salah (Avoid):** "Pemberitahuan darurat Cisco memperingatkan eksploitasi aktif liar atas celah keamanan eskalasi hak istimewa kritis pada fungsionalitas Web UI dari server perangkat lunak IOS XE (CVE-2023-20198)."
- **Benar (Good):** "Cisco mengeluarkan peringatan terkait eksploitasi aktif terhadap kerentanan eskalasi hak istimewa kritis pada fitur Web UI perangkat lunak IOS XE (CVE-2023-20198)."
